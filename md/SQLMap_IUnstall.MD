SQLMap_Install.MD

1. 下载源码：
zip版：https://github.com/sqlmapproject/sqlmap/zipball/master 
tar.gz版：https://github.com/sqlmapproject/sqlmap/tarball/master

2. 安装python环境，需要python2.7 .

3. 解压SqlMap.zip 

4. 运行Cmd 查看python环境 

``` python
python -v

```

5. 切换到SqlMap路径
``` python
python sqlmap/sqlmap.py -help
```
